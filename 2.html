CSS 는 개별적으로 추가 공부하자

css 는 Cascading Style Sheets 이다. 여기클릭

CSS 는 html 을 보다 시각적으로 바꾸는데 사용한다.


html 문법에 tag를 추가한다.
font color= "색상"을 사용하여 다른색깔을 표현한다.
만약에 리스트가 1억개가 넘는다면 이를 어떻게 바꿀수 있을까?
"font color"는 개별적으로 모두 바꿔야 하는 단점이 있다.
